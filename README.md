# CIVE202_Spring2026_Group18-Project3

## CIVE 202 Spring 2026 - Project 3

This GitHub page houses all the deliverable files for the FHWA Data Visualization Project.

### Project Goal:
The goal of this project is to create computer aided visualizations of the NHTS and NGSIM datasets for the Federal Highwat Administration. This will involve the following tasks: 1) Data preparations and organizations, 2) Categorical Visualization, 3) Distribution Analysis, 4) Time-Series Analysis, and 5) an IDM Simulation
